```

### MERCREDI 23 AVRIL — Tip
**Format** : Tip food cost

```
FORMAT TIP FOOD COST #1

Tip : "La perte, ça se calcule."
Contenu : un filet de bœuf perd 12-15% à la parage. Une sole entière perd 45% entre les arêtes et la peau. Si vous achetez à 43€/kg mais n'utilisez que 55%, le coût réel est 78€/kg.

Crée le tip avec ces deux exemples concrets (bœuf et sole). Inclure la formule : Prix réel = prix d'achat ÷ rendement utilisable.
```

### MARDI 29 AVRIL — Stat
**Format** : La stat qui surprend

```
FORMAT STAT QUI SURPREND

Stat : "La règle des 3 tiers : dans un restaurant équilibré, 30% du CA va aux ingrédients, 35% aux salaires, 15% aux charges fixes. Il reste 20% de marge. Dépasser 33% sur les ingrédients, c'est manger directement dans vos salaires."

Crée le post avec cette règle. Visuel : graphique camembert ou barres très simple montrant la répartition idéale.

```

---

### MERCREDI 30 AVRIL — Tip
**Format** : Tip food cost

```
FORMAT TIP FOOD COST #2

Tip : "Ne calculez pas votre food cost qu'une fois par an."
Contenu : l'huile d'olive a pris +180% en 3 ans. Un plat calculé en 2022 avec de l'huile d'olive peut avoir un food cost réel 3-4 points plus haut aujourd'hui. Recalculer = se protéger.

Donner 3 déclencheurs concrets pour recalculer :
1. Changement de carte
2. Variation >10% d'un ingrédient clé
3. Tous les trimestres minimum

Crée le tip. Format visuel : liste des 3 déclencheurs avec icône calendrier/alerte/saison.
```

```

### MERCREDI 14 MAI — Tip
**Format** : Tip food cost

```
FORMAT TIP FOOD COST #4

Tip : "Connaissez-vous vos Stars et vos Poids morts ?"
Contenu : le menu engineering classe chaque plat en 4 catégories.
⭐ Star : populaire + rentable → mettez-le en avant
🐄 Vache à lait : populaire + peu rentable → augmentez le prix de 1-2€
❓ Énigme : rentable + peu commandé → repositionnez-le sur la carte
💀 Poids mort : ni l'un ni l'autre → supprimez-le

Crée le tip avec ces 4 catégories. Visuel : matrice 2×2 simple (popularité / rentabilité). C'est le seul outil de menu engineering dont un restaurateur a besoin.
```
